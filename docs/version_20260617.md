# MCStudio Version Notes - 2026-06-17

## Focus

This update focuses on workstation calibration interaction stability, motion feedback, and controller/test robustness.

## Changes

- Workstation calibration move-sequence buttons now use a stricter state model:
  - Start position is green and clickable before entering the sequence.
  - Current step is blue after motion completes.
  - Only adjacent previous/next steps are green and clickable.
  - Non-clickable steps always use gray borders.
- During motion, all move-sequence buttons are disabled and shown with gray borders to prevent accidental clicks.
- Motion status now immediately shows that the axis or start-position command is moving.
- Workstation calibration axis move wait timeout was increased to 120 seconds.
- Move-sequence visual refresh is centralized so periodic position refresh does not overwrite button states.
- Performance-test buffer abnormal stop handling was improved to show a user-facing warning instead of silently failing.
- Maximum-speed test chart highlights the maximum-speed sample point.

## Notes

- Build output and runtime-generated files are intentionally not part of this version note.
- The exported ACS/PVSpline scan buffers are kept under `Exports/` for handoff and testing.
