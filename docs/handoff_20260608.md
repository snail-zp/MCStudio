## 3. 当前推荐架构

```text
Page
  -> *Service
  -> *ExecutionService
  -> *WorkflowService
  -> ControllerService / Repository
```

说明：

- `Page` 负责 UI 和交互协调
- `*Service` 负责配置读写
- `*ExecutionService` 负责控制器执行
- `*WorkflowService` 负责业务判断和数据组装
